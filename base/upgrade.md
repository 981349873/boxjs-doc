# 如何升级

{% hint style="info" %}
升级脚本后，最好重启一次代理（VPN）
{% endhint %}

### Surge

```bash
首页 > Surge 图标 (左上角) > 外部资源 > 全部更新
```

### QuanX

```bash
长按风车 > 刷新 (左下角) > 重启代理 (VPN)

# 注意：如果长按风车后，左下角没有刷新按钮
# 那么：你应该最近购买的用户，你应该：
点击风车 > 重写 > 引用 > 全部更新 > 重启代理 (VPN)
```

### Loon

```bash
配置 > 插件 > 刷新 (右上角) > 重启代理 (VPN)
```
